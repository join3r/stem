How do I connect Fastmail as an MCP server?

Stem

You can add it from the Manage → MCP tab. Fastmail is a remote server, so it authenticates over OAuth:

  1. Choose Remote and paste https://api.fastmail.com/mcp.
  2. Click Add server, then Sign in to open the browser flow.
  3. Stem restarts the Codex runtime so the server loads.
Note

The chat assistant is sandboxed and can't edit its own config — MCP setup is app-UI only, by design.

You

Will I have to sign in again every time?

Stem

No — the OAuth token is stored in Stem's isolated CODEX_HOME and reused across launches. You'll only re-authenticate if the token is revoked or expires.

If a server stops responding, open Manage → MCP and check its status badge; a quick Sign in refreshes the connection.
